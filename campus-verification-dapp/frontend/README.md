# 🔐 CertiCore

Blockchain-based certificate verification system built on the Core blockchain.

## Problem
Fake academic certificates and transcripts are difficult to verify.

## Solution
CertiCore allows institutions to register and issue certificates on-chain, enabling anyone to verify certificate authenticity.

## Features

- 🔗 Core Wallet connection
- 🏫 Institution registration
- 🎓 Certificate issuance
- ✅ Blockchain verification
- 🔒 Tamper-resistant records

## Tech Stack

- HTML
- CSS
- JavaScript
- Solidity
- Ethers.js
- Core Blockchain
- Core Wallet

## How It Works

1. Institution connects wallet
2. Institution registers certificate details
3. Certificate information is stored on the blockchain
4. Anyone can verify using the certificate ID
